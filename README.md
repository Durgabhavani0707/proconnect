 🚀 **ProConnect**

ProConnect is a full-stack web application that allows users to book home services like cleaning, plumbing, electrical work, and more.


 🌟 **Features**

- 🔐 User Registration & Login
- 🛠 Browse Services (40+ services)
- 📅 Book Services with Date & Time
- 📋 View Bookings
- 🎨 Modern UI (React + Gradient Design)
- ⚡ Fast & Responsive



 🧑‍💻** Tech Stack**

 **Frontend**
- React.js
- CSS (Custom UI)
- React Router

 **Backend**
- Spring Boot
  
 **Database**
- MySQL
